# SOAP Web Service Project

## Description
A SOAP-based web service project for managing DVD rental database (Sakila database). This project provides web services for CRUD operations on various entities including actors, films, countries, cities, languages, categories, stores, staff, and customers.

## Technologies Used
- Java 19
- Jakarta EE 10
- JAX-WS (SOAP Web Services)
- JAXB (XML Binding)
- JPA/Hibernate 6.3.1
- MySQL 8.0
- Maven
- Apache Tomcat 10.1 / Jetty 11

## Features
- SOAP Web Services for:
  - Actor Management (CRUD)
  - Film Management (CRUD)
  - Category Management (CRUD)
  - Language Management (CRUD)
  - City Management (CRUD)
  - Country Management (CRUD)
  - Store Management (CRUD)
  - Staff Management (CRUD)
  - Customer Management (CRUD)
